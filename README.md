# LogoTurtle
I believe this is the First Open Source Logo Programming in Chrome Extension.

# NPM 
The following two classes have been exported to NPM:
- `LogoCanvas` in `js/logocanvas.js`
- `LogoParser` in `js/logoparser.js`
