# Support

For usage questions, bug reports, feature ideas, and translation improvements,
please use the GitHub issue tracker:

- [Bug report](https://github.com/DoctorLai/LogoTurtle/issues/new?template=bug_report.md)
- [Feature request](https://github.com/DoctorLai/LogoTurtle/issues/new?template=feature_request.md)
- [Open issues](https://github.com/DoctorLai/LogoTurtle/issues)

Before opening an issue, please check whether the problem reproduces on the
latest version and include the browser, extension version, operating system,
and a small LOGO program that demonstrates the behavior when possible.

Please report suspected security vulnerabilities through [SECURITY.md](SECURITY.md)
instead of opening a public issue.
